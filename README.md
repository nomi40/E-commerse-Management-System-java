# 🛒 Java E-Commerce Management System

A console-based E-Commerce system built using Java, demonstrating Object-Oriented Programming (OOP), file handling, and real-world system design concepts.

---

# 🚀 Features

## 👨‍💼 Admin Features
- Add new products with auto-generated IDs
- Update product details
- Remove products from system
- View all products
- Persistent storage using file handling

---

## 🛍️ User Features
- View all products
- Search products by ID
- Add products to cart
- View cart with total calculation
- Checkout system with payment options

---

## 💳 Payment System
- Cash on Delivery (COD)
- Online Payment (simulated)
- Discount applied for high-value orders

---

# 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes & Objects
- Interfaces (Payment abstraction)
- File Handling (BufferedReader / BufferedWriter)
- Exception Handling
- Collections (ArrayList)
- Menu-driven system design
- Encapsulation of logic into modules

---

# 📂 Project Structure

JavaProject.java      → Main driver class  
Product class         → Product model  
Cart class            → Shopping cart logic  
Admin class           → Admin operations  
FileHandler class     → File storage system  
Payment interface     → Payment abstraction  

---

# 📁 Data Storage

Products are stored in:

products_info.txt

Format:
ID,Name,Price,Stock

✔ Data persists even after program restarts
